# Startup Navigator

Startup Navigator is a startup execution workspace for teams to manage projects, milestones, tasks, internal updates, and startup KPI progress.

## Documentation

For complete product documentation, see:

- [docs/STARTUP_NAVIGATOR_DOCUMENTATION.md](docs/STARTUP_NAVIGATOR_DOCUMENTATION.md)

## Quick Start

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build production bundle
- `npm run test` - Run tests
- `npm run lint` - Run lint checks
